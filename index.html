Blueprint Cloud Sync Real

Blueprint ini belum menyalakan real-time sync. Ini memetakan struktur Firestore final, aturan konflik, backup, dan rollback sebelum auto-sync penuh dibuat.

Design locked before build
1. User identityData login dan family aktif.users/{uid}
