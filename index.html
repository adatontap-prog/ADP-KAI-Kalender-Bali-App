Tambah / Update Otonan

Catatan penting: pergantian hari Bali mengikuti terbit matahari, bukan jam 24:00. Jika lahir sebelum matahari terbit, sistem mencatat effective Bali date sebagai tanggal sebelumnya. Untuk input Hari Bali langsung, tanggal otonan acuan tetap diperlukan sampai data harian wewaran+wuku lengkap.

Otonan akan dicatat otomatis pada setiap tanggal jatuhnya dalam kalender 2026 sebagai event lokal keluarga, ditandai titik hijau. Data tersimpan lokal di browser HP, belum masuk server.
