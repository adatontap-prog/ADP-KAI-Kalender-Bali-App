Catatan penting: pergantian hari Bali mengikuti terbit matahari, bukan jam 24:00. Jika lahir sebelum matahari terbit, sistem mencatat effective Bali date sebagai tanggal sebelumnya. Untuk input Hari Bali langsung, tanggal otonan acuan tetap diperlukan sampai data harian wewaran+wuku lengkap.

Otonan akan dicatat otomatis pada setiap tanggal jatuhnya dalam kalender 2026 sebagai event lokal keluarga, ditandai titik hijau. Data tersimpan lokal di browser HP, belum masuk server.

Kontrol Data Lokal

Backup belum dibuat pada perangkat ini. Data otonan masih lokal di HP/browser.

Untuk mockup publik gunakan placeholder: Anggota Keluarga 1, Anggota Keluarga 2, Anak 1, User 1. Jangan tampilkan identitas asli saat demo publik.

Catatan validasi: data awal berstatus digital_source_only. Untuk keputusan adat/upacara penting, tetap verifikasi dengan kalender Bali cetak/resmi, sumber adat setempat, atau pemangku. Otonan pada Phase 1.8 mendukung input Gregorian/Hari Bali langsung, backup data lokal, export jadwal otonan 2026, export kalender ICS, dan display feed JSON. Pergantian hari Bali memakai prinsip terbit matahari; gunakan verifikasi keluarga/adat untuk keputusan penting. Pusat reminder membantu persiapan H-30/H-7/H-3/H-1/Hari-H; backup/export membantu menjaga data lokal; ICS membantu sinkron ke kalender umum; display feed JSON menjadi jembatan awal ke tablet/ESP32/e-paper.
